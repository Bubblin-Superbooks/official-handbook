Bash Your Book

Feel free to skip this section and move on to the code playground (book editor) if you're in a hurry. But if you have a kick to learn a few new things then it's recommended to go through this step right now. Will help you save the day later.

This will take about 30 minutes.

Linux users usually know about it but if you're on mac then we're going to use what is called the terminal app. Or the bash or the shell.

Launch the terminal (learn how) on your computer and it'll show a prompt like this:

~/projects/ marvin$: _

Unfortunately, users on windows do not have access to a unix-style shell interface so you might have to install Babun or cygwin before the next step.

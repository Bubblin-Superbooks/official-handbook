Step-1: Installing Homebrew

First, install Homebrew.

Homebrew allows you to install and compile necessary software packages easily from their source.

It comes with a handy little installation script:

$ ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"

Run (paste) the command above in your terminal. When it asks you to install XCode CommandLine Tools, say yes.

Once the installation is complete…

$ brew update

$ brew doctor

…it will say Your system is ready to brew on the console.

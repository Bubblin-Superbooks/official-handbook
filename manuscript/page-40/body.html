Intrinsic Design

CSS has a very important function for your book: layout. It styles the content of your book and gives it a personality. On Bubblin every page of a book is a virgin iframe you can apply CSS to:

  1. Via CSS template for the entire book, or
  2. Via page level CSS for isolated control.

Before writing your first CSS rule please take a look at Bubblin's prescribed standard for Superbook layouts. Do not use heavy CSS frameworks like bootstrap or normalize inside, it will not work.

Refer our free template for intrinsic design.

Role of CSS doesn't stop with layouts. CSS can also become content itself. Take a look at our Sleepy Parakeet on the next page that is made up entirely of CSS.

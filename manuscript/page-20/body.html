To create a new comic book my-awesome-new-comics with say 24 pages:

$ bookiza new my-awesome-new-comics --leafs 12 --template comics

Bookiza will scaffold a new project with 12 fresh leafs (24 pages) inside the manuscript/ folder and apply the template layout of a comics.

It will then proceed to install all the nodejs dependencies for your project and then seek your confirmation to commit the project into a Github repository. Type yes when it does.

Boom!

CD into my-awesome-new-comics, and start:

$ bookiza server

JavaScript

There are four ways to add JavaScript to your page:

  1. via per-page HEAD inserts,
  2. via textarea to hold script,
  3. via body of HTML,
  4. via a HEAD or Script template.

It is best to include JS only on specific pages where it is required. This lowers the processing overhead onload of the superbook into the dom. It also eases debugging when you exactly know which page(s) to look into for JavaScript.

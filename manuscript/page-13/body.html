Bookiza Framework

This will take about 90 minutes to set up.

Linux users or developers usually know about this but if you're on Mac then we're going to use what is called the terminal app, or the bash, or the shell to get started.

Learn (how to use a terminal.)

On the terminal, you'll see a prompt:

~/projects/ marvin$: _

The prompt is where you're going to type in or copy-paste the following commands to complete installation. That's why the name 'command-line' or console. ;-)

Unfortunately, users on windows do not have access to a unix-style shell interface. You'll have to install Babun or Powershell or Cmder or Cygwin and come back to this page later with a running terminal on your computer.

Book Name & URL

The book name or the project name is built into the url of a published book. It is very easy to name a book on Bubblin. Click on Untitled link on top of the editor and provide the name through overlay (Screenshot):

The url of your book will be like so:

https://bubbl.in/book/My-Awesome-Book-Name-by-Author-Name

If you're writing the book in your local using Bookiza then the project name must be the same as the book name.

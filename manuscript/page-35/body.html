The code editor will automatically combine the HTML, CSS, JS and HEAD components and render it on ActivePage. Note, you can write the entire book from your browser itself, but we recommend you to do it from your local machine instead, using the Bookiza Framework as explained in the previous section.

To toggle ActivePage click on the page-numbers listed for editing on the toolbar.

For example, clicking on [2] will activate page-2 (will turn green) and then the code playground will render output on page-2 instead.

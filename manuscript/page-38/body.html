Using the HEAD

Bubblin Editor interface allows for the ability to include CDN resources via the HEAD of your book. It is rather one of the best features of a Superbook. There are two ways to add CDN resources into the HEAD of your page:

  1. Via Superbook template, or
  2. Via page level HEAD component.

Templates are ideal for configuration of the entire book.

For example, if you're writing a book on mathematics or on elementary particle physics then it's likely you're going to need Mathjax (Latex) on all of the pages. Add the MathJax library from the template HEAD.

If the need is specific to a page then you can include that resource via page level HEAD on the Bubblin Editor.

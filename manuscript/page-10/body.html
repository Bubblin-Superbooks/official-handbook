Set up

First thing you need to be able to publish is a Bubblin account. Sign up now if you haven’t already.

Once you've signed up and logged into Bubblin, press the keys ⌘ / CTRL + U to open a menu on your left. Click → New to open a new project with a code playground around it.

Feel free to jump to this quick tutorial now to see how to write a Superbook using the in-browser code playground. No further setup is required.

The online editor is enough for short novellas, magazines or comic books i.e. tiny packages.

For large projects like a textbook, photobook or an interactive story or for commercial publishing it is recommended to install and use Bookiza instead.

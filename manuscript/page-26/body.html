Bookiza will ask for a Github URL next. Head over to https://github.com and create a new repository using the same hyphenated-name as your project's. Give a one-line description to it too.

Leave the "Initialize this repository with a README" checkbox unchecked and click on Create Repository to get a Github URL:

Copy the HTTPS url from Github and provide it to Bookiza (paste) on your terminal. Ready! A barebones bookiza project is ready for some love now.

Note we've used a public repository over here but you can choose private repositories as well if you want to keep your manuscript hidden from the world.

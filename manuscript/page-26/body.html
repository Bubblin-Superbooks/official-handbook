Step 5: Installing Bookiza

Bookiza is a simple Command Line utility (CLI) and a stripped down application framework designed to manage your manuscript.

To set it up:

$ git clone --depth=1 https://github.com/bubblin/bookiza.git ~/.bookiza

$ echo "source ~/.bookiza/.bookiza" >> ~/.bash_profile

Typing $ bookiza -v will output the following:

$ bookiza version 0.1.0 # Or later

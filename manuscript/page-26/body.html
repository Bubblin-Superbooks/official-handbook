Bookiza will ask for a Github URL next. Head over to https://github.com and create a new repository there. Use the same hyphenated-name as your project name and give it a one-line description too.

Leave the "Initialize with a README" option unchecked because bookiza will do that for us:

Copy the repo_url (HTTPS) from Github and provide it to bookiza, i.e paste it on your terminal. That's it. A fresh new book project is ready for some love. Type:

$ bookiza help

to see available methods. Note that we've used a public repository for this example, but you can choose to use a private one to hide your manuscript from the world.

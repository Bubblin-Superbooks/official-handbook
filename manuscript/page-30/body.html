It quickly provides all the project defaults and gives a structure to our manuscript so that we can focus on writing our book. For the most part, we'll be working inside the manuscript/ folder of the project. That's where all of our content / code would go.

Inside the manuscript/ there are pages of the book. Since Bubblin uses web as the default medium for its books, our manuscript is essentially a sum of on three components: The building blocks of web: HTML, CSS & JavaScript.

Inside each page folder you'll find the following:

  1. body.html
  2. style.css

Sometimes there will also be a script and a head compment inside a page. Like so:

  1. script.js
  2. head.html

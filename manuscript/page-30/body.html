Head over to https://github.com to create a new repository for our book. Use the same hyphenated-name as your projectname and give it a one-line description too. Leave the "Initialize with a README" option unchecked because bookiza will do that for us.

Copy the repo_url (HTTPS) from Github and feed it to Bookiza, i.e paste it on your terminal. That's it.

A fresh new project is ready for some love. Type:

$ bookiza help

to see the methods available to you from Bookiza.

Note we've used a public Github repository for this example but you can use a private one to hide your manuscript from the world.

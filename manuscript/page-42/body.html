JavaScript

Since a Superbook lives on the web, we can add JavaScript into the pages for fancy effects in one of the four following ways:

  1. Via page specific HEAD loader,
  2. Via Editor Textarea for JavaScript,
  3. Via body of HTML Textarea on Editor, and
  4. Via HEAD template for the entire book.

It is recommended to include JavaScript only on pages where it is necessary. This will lower the processing overhead of your Superbook on devices that are less powerful and improve battery performance.

Restricting JS to only specific pages also helps in debugging.

Step 4: Installing Bookiza

Bookiza is a simple Command Line Interface (CLI) to help you manage your manuscript efficiently.

To set it up:

$ git clone --depth=1 https://github.com/bubblin/bookiza.git ~/.bookiza

$ echo "source ~/.bookiza/.bookiza" >> ~/.bash_profile

Typing $ bookiza -v will output the following:

$ bookiza version 0.1.0 # Or later

Now for the moment of truth, let's create our first book project with say 12 pages.

$ mkdir Books && cd Books

$ bookiza new YOUR-BOOK-NAME 12

Pro tip: Use hyphenated strings to name your project, e.g: My-Own-Harry-Potter.

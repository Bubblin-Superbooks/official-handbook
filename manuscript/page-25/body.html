Default Editor Format (on web):

Editor on Bubblin website processes only the following format combination:

"mode": { "HTML": "html", "CSS": "css", "JS": "js", "HEAD": "html" }

The decision to use only HTML for book writing on web is intentional. If you prefer to use markdown (or any other preprocessor) we recommend you install bookiza on your local with your desired flavor .bookrc like so:

"mode": { "HTML": "markdown", "CSS": "less", "JS": "TS", "HEAD": "haml" }

Any combination of markdown, haml, pug, sass, less or stylus will work. Refer documentation for further information.

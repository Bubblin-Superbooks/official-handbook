To toggle the ActivePage click on the page-numbers listed on the toolbar.

For example, clicking on [2] will activate page-2 (will turn green) and then the code playground will render output on page-2 instead.

The Code Playground will automatically combine your HTML, CSS & JavaScript and render it on the ActivePage. Note, it is possible to write the entire book from your browser itself but it is not recommended to do so.

We recommend using the Bookiza Framework to compose and manage your manuscript from your local.

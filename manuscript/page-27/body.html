Now for the moment of truth:

Let's create a project named MY AWESOME BOOK NAME with say 12 pages.

$ mkdir projects && cd projects/

$ bookiza new MY-AWESOME-BOOK-NAME 12

Bookiza will setup all directories and create all the necessary files for our book manuscript. It will then seek to check-in the project into Github, a repository.

Pro tip: Use hyphenated strings to name your project.

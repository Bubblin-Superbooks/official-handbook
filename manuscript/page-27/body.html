Now for the moment of truth:

Let's create a project named MY AWESOME BOOK NAME with say 24 pages.

$ mkdir projects && cd projects/

$ bookiza new MY-AWESOME-BOOK-NAME 24

Bookiza will setup all directories and create all the necessary files for our manuscript. Inside the manuscript/ folder it will also create 24 blank pages that we asked Bookiza to create for us.

Bookiza will ask for a Github URL next.

Pro tip: Use hyphenated strings to name your project.

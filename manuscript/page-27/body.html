Layouts

It's very easy to apply a layout to a Superbook

Use the templates feature to add a layout (style) that works across the book. Occassionally, you may also need a template for HTML, JavaScript & HEAD be applied all along your book. For any of that, use the dropdown for templates…

…as shown below, on the Bubblin website:

On a Bookiza project layout can be added via the templatesfolder ($ [projectname] > templates/) of your project. Bookiza will automatically apply the template to your book. Check out some free and open source templates on Github to get started.

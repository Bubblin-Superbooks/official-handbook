JavaScript

There are four ways to add JavaScript to your page:

  1. via per-page HEAD inserts,
  2. via textarea to hold script,
  3. via body of HTML,
  4. via HEAD /Script template.

It is recommended to include JS only on specific pages where it is required. This would lower the processing overhead onload of your superbook into the dom. It also helps in debugging when knowing exactly which page(s) to look into.

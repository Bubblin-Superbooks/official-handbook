The code editor will automatically combine the components you provide in HTML, CSS and JS fields and render it on the ActivePage.

You can toggle ActivePage by clicking on the page numbers listed for editing on the toolbar.

In the image above, note both page-2 and page-3(active) are available for viewing. Clicking on [2] will activate page-2 (turn green) and then the code editor will output into page-2 on the left.

Mode: Full HTML5 editor mode is available.

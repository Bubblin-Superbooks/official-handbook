Step-2: Installing GIT

Next install git-scm to manage versioning and history of your project (book). Type the following command in your terminal.

$ brew install git

To verify:

$ git --version

It will say git version 2.5.1 or later on your console.

You can also use other scm tools like Mercurial, Subversion or Perforce for source code management. For the purpose of this book we will go with Git because it is so nice to work with.
